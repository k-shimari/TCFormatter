# TCFormatter
.txt形式で出力された縦に長い出力を，csv形式に直すツールです．
必要のない行をスキップすることも部分的に可能です．

例：
所属，名前，身長，体重がN人分書かれた入力ファイル`data.txt`がある．
そこから，名前，身長，体重を取り出して縦 * 横が3 * Nの`data.csv`に直す．

入力：`data.txt`

    1-A
    Aさん
    170
    60
    1-A
    Bさん
    175
    70
    ...

`java -jar TCFormatter.jar /path/to/data.txt 1 3`

  * 第一引数：入力のテキストファイルへのパス
  * 第二引数：データごとにスキップする行数（今回は1-Aの1行ををスキップするため1）
  * 第三引数：データごとに処理する行数（今回は名前、身長，体重の3つを処理するため3）

出力：`data.csv`（data.txtと同じディレクトリに出力）

    Aさん, Bさん, ...
    170, 175, ...
    60, 70, ...
